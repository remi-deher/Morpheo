# Welcome to the Morpheo documentation

Morpheo is a universal synchronization orchestrator for .NET.

> [!TIP]
> To get started, check out the [Installation Guide](https://github.com/remi-deher/Morpheo) or explore the **[API Reference](api/index.html)**.

## Learn more

The project is divided into two parts:

* **[Morpheo.Core](api/Morpheo.Core.html)**: The synchronization engine.
* **[Morpheo.Sdk](api/Morpheo.Sdk.html)**: The development kit for extensions.
