    Class MorpheoMetrics

    Centralised metric instruments for a Morpheo node, built on the BCL System.Diagnostics.Metrics API. Any OpenTelemetry exporter (or dotnet-counters) can subscribe to the meter named MeterName without Morpheo taking a dependency on the OpenTelemetry SDK.

    To export with OpenTelemetry, the host app simply does: 

    services.AddOpenTelemetry().WithMetrics(m => m.AddMeter(MorpheoMetrics.MeterName));
