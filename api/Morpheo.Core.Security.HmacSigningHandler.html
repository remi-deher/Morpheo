    Class HmacSigningHandler

    Client-side counterpart of PskHmacAuthenticator. Signs every outgoing inter-node request with the cluster secret so the receiver can authenticate it.

    For each request with a body it sets three headers:

    • X-Morpheo-Timestamp — unix milliseconds, for freshness.
    • X-Morpheo-Nonce — a random value, for replay rejection.
    • X-Morpheo-Signature — HMAC-SHA256 over {timestamp}.{nonce}.{body}, hex-encoded.

    If no cluster secret is configured the handler is a transparent pass-through. It must run before any request-compression handler so the signature covers the plaintext body — the same bytes the server sees after decompression.
