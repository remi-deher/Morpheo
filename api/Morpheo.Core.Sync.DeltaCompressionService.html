    ApplyPatch(string, string)

    Applies a patch to an original JSON document to produce the modified version.

    Parameters
    Type Name Description
    string originalJson

    The original valid JSON string.
    string patchJson

    The patch JSON string to apply.
    Returns
    Type Description
    string

    The resulting transformed JSON string.
    Remarks

    If the patch is an empty JSON object "{}", the method interprets it as "no change" and returns the original JSON. Otherwise, it treats the patch as the complete new state (Full Replacement).

    Type Condition

    Thrown when either input (original or patch) is null, empty, or not valid JSON.

    CreatePatch(string, string)

    Generates a patch representing the difference between an original JSON document and a modified version.

    Declaration
    public string CreatePatch(string originalJson, string modifiedJson)
    Parameters
    Type Name Description
    string originalJson

    The original valid JSON string.
    string modifiedJson

    The modified valid JSON string.
    Returns
    Type Description
    string

    A JSON string representing the patch needed to transform the original into the modified version.
    Remarks

    This implementation currently uses a "Full Replacement" strategy where the patch is simply the new content, unless the content is identical. If the documents are identical, an empty JSON object "{}" is returned.

    Exceptions
    Type Condition
    ArgumentException

    Thrown when either input (original or modified) is null, empty, or not valid JSON.
